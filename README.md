## Geometry Functions

### `calculate_pyramid_volume(base_area: float, height: float) -> float`

Calculates the volume of a pyramid using the formula `(1/3) * base_area * height`.

**Parameters:**
- `base_area` (float): The area of the pyramid's base. Must be non-negative.
- `height` (float): The perpendicular height of the pyramid. Must be non-negative.

**Returns:**
- `float`: The calculated volume of the pyramid.

**Raises:**
- `ValueError`: If `base_area` or `height` are negative.
