# Guess the Number Game

This document provides instructions on how to run and play the "Guess the Number" game.

## How to Run the Game

To play the "Guess the Number" game, follow these simple steps:

1.  **Ensure Python is installed:** Make sure you have Python 3.x installed on your system. You can download it from [python.org](https://www.python.org/downloads/).
2.  **Navigate to the project directory:** Open your terminal or command prompt and change your current directory to where the game's Python script (e.g., `guess_the_number.py`) is located.
